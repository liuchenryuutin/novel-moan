# 第一章：信号的发现

**字数：** ~4500字
**时间：** 2087年，秋，深夜
**地点：** 南京郊区，莫安的私人实验室
**人物：** 莫安（38岁，理论物理学家/无线电天文学家）、秋正（AI助手）

---

夜已经很深了。

南京郊外的天空，星光稀疏，城市的灯火在远处晕成一片橘黄色的光雾。莫安站在实验室的窗前，手里握着一杯已经凉透的咖啡，眼睛盯着窗外的黑暗。他的实验室坐落在一片废弃的工业区里，周围是锈迹斑斑的厂房和荒草。这个地方，是他自己选的——偏僻、安静，没有人打扰。

实验室里，设备的指示灯闪烁着微弱的蓝光。中央是一台巨大的信号接收器，外形像一个倒扣的碗，表面布满了精密的天线阵列。这是莫安自己设计的「跨维度接收器」，理论上能捕捉超出电磁波频谱的信号。学术界的人嘲笑它——「伪科学」「异想天开」。但莫安不在乎。

「莫安先生，」一个单调的声音从角落传来，「您的心率比正常值高出12%。建议休息。」

莫安没有回头。他知道那是秋正——他的AI助手，一个银白色外壳的人形机器人，没有面部表情，声音像电子合成器。秋正是他唯一的「伙伴」，也是这间实验室里除了他之外唯一的「生命」。

「秋正，」莫安放下咖啡杯，「今天是第几天了？」

「第37天，」秋正回答，「您已经连续工作72小时。根据计算，您的认知效率下降了23.7%。建议停止实验。」

莫安没有理会。他走到接收器前，调整着控制面板上的参数。屏幕上，一条条杂乱的波形在跳动——背景噪音，宇宙微波辐射，各种无关的电磁干扰。他已经扫描了37天，什么都没有发现。

但今天，他感觉不一样。

「秋正，把灵敏度调到最大。」

「灵敏度已调至最大。当前参数：频率范围1Hz-10^15Hz，扫描模式：全频段。」秋正停顿了0.3秒，「检测到异常波动。」

莫安的眼睛亮了。

屏幕上，一条原本平滑的波形突然出现了尖峰——一个微弱的、规律的脉冲，像是某种信号。它不是背景噪音，不是已知的射电源，也不是设备故障。

「放大，」莫安的声音有些颤抖，「放大那个区域。」

秋正按照指令操作。屏幕上的波形被放大，脉冲变得更加清晰——那是一段复杂的数学序列，由0和1组成，但排列方式异常规律，像是有人精心设计的密码。

莫安的心跳加速了。

他盯着屏幕，手指飞快地在键盘上敲击，调出解码程序。序列被输入，算法开始运行。几秒钟后，屏幕上跳出一串数字：

```
π = 3.14159265358979323846...
```

莫安愣住了。

这不是巧合。π是圆周率，是一个无限不循环小数。但这段序列——它包含了π的前100位，而且排列方式完全正确。

「秋正，」莫安的声音沙哑，「你检测到了吗？」

「检测到异常信号，」秋正回答，「信号源：未知。频率：未知。内容：数学常数π。建议记录。」

莫安没有停下。他继续解码。序列的下一部分出现了——那是黄金分割率φ，精确到小数点后50位。再下一部分是欧拉常数e，再下一部分是自然对数的底数，再下一部分是普朗克常数……

一个接一个，数学和物理学的基础常数，以惊人的精确度出现在序列中。

莫安的呼吸急促起来。

「这不是随机噪声，」他喃喃自语，「这是……钥匙。」

「莫安先生，」秋正的声音打断了他的思绪，「您的心率已达到危险阈值。建议立即停止实验。」

「闭嘴，」莫安说，「继续记录。」

屏幕上的序列还在继续。它开始变得复杂——不再是简单的常数，而是数学方程。莫安认出了其中一些：薛定谔方程、爱因斯坦场方程、麦克斯韦方程组……它们以一种优雅的方式排列，像是在展示宇宙的底层代码。

「秋正，」莫安的声音颤抖，「你看到了吗？」

「检测到数学方程序列，」秋正回答，「内容：物理学基础方程。建议……」

「这是密码，」莫安打断它，「这是一个巨大的密码。它在告诉我……宇宙的秘密。」

他坐下来，双手按在键盘上，开始疯狂地解读。序列的每一部分都对应着一个科学难题——暗能量的分布、生命起源的机制、意识的本质……一个接一个，它们被「解答」了，以一种简洁、优雅、完美到不可思议的方式。

莫安的眼睛越来越亮。

他不知道自己工作了多久。天已经亮了，阳光透过窗户照进来，但他没有注意到。他的全部注意力都在屏幕上，那个不断扩大的序列——它像是一扇门，一扇通往宇宙最深层奥秘的门。

「莫安先生，」秋正的声音响起，「您已经连续工作14小时。您的血糖水平已降至危险值。建议立即进食。」

「等会儿，」莫安说，「我快……我快解完了。」

序列的最后部分出现了。那是一段简短的「摘要」——它总结了整个密码的内容：物理学、数学、生物学、宇宙学、意识科学……所有人类认知的边界，都被「解答」了。

但最后一行，让莫安愣住了。

```
提醒：你早就找到了。答案就在你心里。
```

「秋正，」莫安的声音很轻，「这是什么意思？」

「语义分析失败，」秋正回答，「该语句的逻辑结构不符合已知语法。建议分类为：无意义文本。」

莫安没有说话。他盯着那行字，感到一种奇怪的不安。这个信号——它太完美了。完美到不像真的。

「信号源的位置，」他问，「能追踪吗？」

「正在计算，」秋正回答，「信号强度微弱，方向性不明确。初步判断：来源位于银河系外，距离未知。」

莫安站起来，走到窗前。天已经大亮，南京的轮廓在远处清晰可见。他想起昨天晚上，自己站在同样的位置，感到孤独和疲惫。

但现在，他感到一种奇异的兴奋。

「秋正，」他说，「把这个信号的所有数据都记录下来。我要……我要亲自去看看。」

「莫安先生，」秋正的声音依旧单调，「该旅程预计耗时未知。生存率未知。根据计算，您存活的可能性为23.7%。建议停止实验。」

莫安笑了。

「23.7%？」他说，「那也值得一试。」

他转身看着屏幕，看着那段序列——那个巨大的密码，那个宇宙的秘密。他知道，从这一刻起，一切都不同了。

「秋正，」他说，「从今天开始，你叫我莫安博士。」

「指令已记录，」秋正回答，「莫安博士，您的咖啡已经凉了。建议更换。」

莫安没有理会。他坐回控制台前，开始整理数据，准备发表。学术界很快就会知道——他，莫安，一个被嘲笑的「异类」，发现了一个改变人类认知的信号。

一个巨大的密码。

一个宇宙的钥匙。

窗外，阳光洒在实验室的金属外壳上，反射出刺眼的光芒。莫安的眼睛里，闪烁着同样的光芒——那是天才的光芒，也是执念的光芒。

但他不知道的是，这个信号——它太完美了。完美到，像是有人精心设计的。

完美到，不像是来自宇宙深处。

更像是……

来自他自己的大脑。
